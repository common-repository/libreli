Libreli is a simple plugin that helps you manage your books.

== Description ==

Wouldn’t it be nice if you had an automatically generated and always up-to-date list of the most popular retail sites selling your books?  Libreli is the simple WordPress plugin that does just that.

*   Automatic updates of US and international stores
*   Only lists the stores that have your book in stock
*   Notifications when new stores carry your book
*   Customize the color to go with your book design
*   Generate more income by using your affiliate accounts (premium version)

https://vimeo.com/559504980

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. Log in to your WordPress dashboard, go to the Plugins menu and click Add New.

Type "Libreli" and click Search Plugins. Once you’ve found this plugin you can install it by simply clicking “Install Now”.

= Manual installation =

To manually install the plugin downloading the plugin and upload it to your web server via your favorite FTP client application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Getting Started ==

* The plugin will create a new post type called Libreli Books
* You can simply add new books as if you would be creating new posts by entering the ISBN for your book
* On the Libreli Book Manager page you can see all the books you added and their available shortcodes
* Put the shortcode on any page to display a list of stores where your book is available
* The list of stores updates automatically (daily)

== Frequently Asked Questions ==

= Where Can I find the shortcode for the book I created

You can find all the shortcodes for your books at Libreli Books > Libreli Book Manager
